## Quickstart

On **x86 Linux** systems, run

```bash
curl -s https://gist.githubusercontent.com/BlueGummi/beca806b9339c4d934260d1eaf3d84cb/raw/32e89f0e8c4c88867e9faae77d2587e67690d8b6/binstall.sh | bash
```

On **macOS and other Unix systems**, run

```bash
git clone https://github.com/BlueGummi/belle --depth=1 && cd belle && ./build.sh -w && ./install.sh -c
```

###### AUR Installation - Binary

```bash
yay -S belle-cpu
```

And on Windows, run

```pwsh
iex ((New-Object System.Net.WebClient).DownloadString('https://gist.githubusercontent.com/BlueGummi/7ebc6a09cbf39cc88304cb8a8d8bb571/raw/0102d27771f9080ecd6fc91b5879f6bcb8ddde51/binstall.ps1'))
```

The binaries can be run by calling `basm`, `belle`, or `bdump`.

#### Inspiration

[The Little Computer 3](https://en.wikipedia.org/wiki/Little_Computer_3) - Inspired some of the design aspects of the BELLE-ISA and BELLE, particularly word-addressed memory and the 16-bit nature of the machine (it's why "BELLE" has two L's - it's to honor the LC3)

[Intel x86 assembly](https://en.wikipedia.org/wiki/X86_assembly_language) - Inspired the syntax for the assembly.
